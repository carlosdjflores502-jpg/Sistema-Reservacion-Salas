# Sistema de Reservación de Salas y Recursos

## Descripción del proyecto
El Sistema de Reservación de Salas y Recursos es una aplicación web orientada a estudiantes, docentes y personal administrativo, que permitirá reservar salas, laboratorios y recursos tecnológicos de forma organizada y controlada.

## Problemática identificada
Actualmente, las reservas se realizan de forma manual o por medios informales, lo que genera los siguientes problemas:

- Doble asignación de espacios
- Conflictos de horarios
- Falta de control sobre los recursos
- Pérdida de tiempo en la coordinación

## Objetivo general
Desarrollar un sistema digital que permita gestionar de forma eficiente la reservación de salas y recursos dentro de la institución.

## Metodología seleccionada
Para el desarrollo del proyecto se seleccionó la metodología Scrum, debido a que permite trabajar de forma iterativa, organizada y flexible, facilitando la entrega de avances por etapas.
